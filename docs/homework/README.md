# 实现作业目录

这个目录用于承接 `study` 学完后的编码作业。

## 目录职责

- `study`
  - 解决“是否真的理解了”
- `reviews`
  - 解决“我的理解哪里还不准”
- `design`
  - 解决“在当前 Kratos 仓库里应该怎么落”
- `homework`
  - 解决“下一步具体先写什么”

## 当前状态

- 已有作业：`docs/homework/01-config-round-1.md`
- 已有作业：`docs/homework/01-config-round-2.md`
- 当前主题：`Config get / listen`

## 使用原则

- `homework` 不承担学习过程答题功能
- 学习过程先放 `reviews`
- `homework` 只保留最终要实现的任务和验收标准
