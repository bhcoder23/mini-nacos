# 实现评审记录

这个目录用于记录每一轮 `mini-nacos` 实现后的 `Code Review`、验收意见和复盘。

## 目录职责

- `study` 负责理解校验
- `homework` 负责实现任务定义
- `reviews` 负责 CR 和验收留痕

## 命名建议

- `docs/reviews/2026-04-29-naming-round-1.md`
