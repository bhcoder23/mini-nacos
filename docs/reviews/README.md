# 学习评审记录

这个目录优先用于记录 `mini-nacos` 的学习过程，不只是实现后的 `Code Review`。

## 目录职责

- `study`
  - 放整理后的最终学习笔记
- `reviews`
  - 放学习过程里的答题记录、点评、追问、是否过关
- `design`
  - 放理解源码后，映射到当前仓库的设计稿
- `homework`
  - 放最终编码任务、验收标准、CR 清单

## 推荐工作流

1. 先看 `study` 给出的阅读范围
2. 再把自己的理解写进 `reviews`
3. 由我对 `reviews` 里的答案做点评和追问
4. 过关后，再把结论沉淀回 `study`
5. 学完整个主题，再进入 `design` 和 `homework`
